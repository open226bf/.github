# Gouvernance — OPEN226

Ce document décrit comment les décisions sont prises au sein de la communauté OPEN226.

---

## Modèle Progressif

La gouvernance d'OPEN226 évolue avec la croissance de la communauté.

### Phase 1 — Fondation (actuelle)

**Structure :** Leadership unique du fondateur.

- Le fondateur prend les décisions stratégiques
- Les contributions et suggestions sont bienvenues via les Issues et Discussions
- Toutes les décisions importantes sont documentées publiquement

### Phase 2 — Comité Consultatif (à partir de 5 membres actifs)

- Mise en place d'un comité consultatif informel
- Les décisions majeures sont discutées collectivement avant d'être tranchées
- Introduction des votes pour certaines décisions techniques

### Phase 3 — Gouvernance Élue (à partir de 15 membres actifs)

**Rôles élus :**

| Rôle | Responsabilités |
|------|----------------|
| Présidente/Président | Direction stratégique, représentation externe |
| Secrétaire | Communication, comptes-rendus, coordination |
| Responsable Technique | Architecture, revue de code, standards |

- Les élections ont lieu tous les 12 mois
- Toute personne ayant au moins 3 contributions acceptées peut être candidate
- Le vote est ouvert à tous les membres actifs

---

## Processus de Décision

### Décisions courantes
Prises par le fondateur ou les mainteneurs. Commentaires bienvenus via les Issues.

### Décisions importantes
(Changements de licence, orientations stratégiques, nouveaux projets majeurs)
- Ouverture d'une Discussion pendant 7 jours minimum
- Résumé des avis et prise de décision documentée

### Amendements à la gouvernance
- Proposition via une Issue avec le label `gouvernance`
- Discussion communautaire de 14 jours
- Approbation par consensus ou vote à la majorité simple

## Niveaux de Contribution

| Niveau | Critères | Droits |
|--------|----------|--------|
| **Explorateur** | Inscription et première interaction | Accès aux ressources et discussions |
| **Contributeur** | 3+ contributions acceptées | Badge, mention dans le README |
| **Mainteneur** | Contributions régulières + reviews | Droits de merge, mentorat |
| **Ambassadeur** | Organisation d'événements, représentation | Invitations conférences, réseau |
