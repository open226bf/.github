## 🌍 À propos

**OPEN226** est une communauté burkinabè dédiée à la promotion et au développement de solutions open source adaptées au contexte local. Ce dépôt est le cœur de la communauté : charte, roadmap, discussions et ressources pour tous les membres.

---

## 📂 Structure de l'organisation

| Dépôt | Description |
|-------|-------------|
| [`.github`](https://github.com/open226bf/.github) | Profil public de l'organisation |
| [`open226-community`](https://github.com/open226bf/open226-community) | Ce dépôt — charte, roadmap, discussions |
| `open226-website` *(à venir)* | Site web officiel de la communauté |
| *Projets pilotes* *(à venir)* | Solutions open source locales |

## 🚀 Démarrage rapide

### Rejoindre la communauté

1. **Lisez** notre [Code de Conduite](CODE_OF_CONDUCT.md)
2. **Consultez** la [Feuille de Route](ROADMAP.md)
3. **Contribuez** en suivant le [Guide de Contribution](CONTRIBUTING.md)
4. **Participez** aux [Discussions](https://github.com/open226bf/open226-community/discussions)

### Proposer un projet

Ouvrez une [Issue](https://github.com/open226bf/open226-community/issues/new) avec le label `💡 idée-projet` en décrivant :
- Le problème que vous souhaitez résoudre
- Le public visé
- Les technologies envisagées

## 🗺️ Feuille de Route

### Phase 1 — Fondation (0-3 mois) ← *Nous sommes ici*

- [x] Créer l'organisation GitHub
- [x] Rédiger le Document de Vision
- [x] Créer le Guide de Contribution
- [ ] Mettre en ligne le site web
- [ ] Lancer 1-2 projets pilotes
- [ ] Première réunion communautaire

### Phase 2 — Croissance (3-6 mois)

- [ ] Recruter 3-5 contributeurs réguliers
- [ ] Organiser le premier atelier de formation
- [ ] Premier hackathon mensuel
- [ ] Créer les groupes de travail thématiques

### Phase 3 — Consolidation (6-12 mois)

- [ ] Élire un comité directeur
- [ ] Partenariats avec universités et entreprises
- [ ] Programme d'incubation de projets
- [ ] Première conférence Open Source BF
